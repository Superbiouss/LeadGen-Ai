Instructions

Connecting your Lead Vault

To see real leads in the "Leads Vault", create a separate n8n workflow:

  1. Add a Webhook node (Method: GET, Path: leads-api).
  2. Add a Google Sheets node to Get Many from your Leads sheet.
  3. Add a Respond to Webhook node to return the data.
  4. Paste the production URL in the "Leads Data API URL" field above.
