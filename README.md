# LeadGen AI: Premium Automation Dashboard

A high-octane dashboard for monitoring and triggering N8N lead generation workflows. Built with a gritty, high-contrast aesthetic.

![Dashboard Overview](https://via.placeholder.com/1200x600?text=LeadGen+AI+Dashboard+Overview)

## ⚡ Features

- **Forge Aesthetic**: Sharp corners, high-contrast palette (`Forge Black` & `Combat Red`), and elite typography (**Bebas Neue** & **Barlow**).
- **Real-time Monitoring**: Monitor extraction accuracy, verified emails, and live activity logs.
- **Workflow Trigger**: Streamlined interface to launch N8N scraper tasks with custom parameters.
- **Analytics Suite**: Interactive growth charts and source distribution data.
- **Responsive Design**: Optimized for desktop and mobile performance.

## 🚀 Getting Started

### Prerequisites

- An active **N8N** installation.
- A configured N8N workflow (see [Workflow Setup](#workflow-setup)).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/LeadGenerator-N8N.git
   ```
2. Open `index.html` in your favorite browser.

For a detailed, step-by-step walkthrough of setting up N8N and Google Maps, check out the **[SETUP_GUIDE.md](file:///home/aakashyadav/Antigravity_Projects/LeadGenerator-N8N/SETUP_GUIDE.md)**.

## 🎨 Design System

This project follows a professional, high-contrast design system:
- **Primary Color**: `#e8192c` (Signature Red)
- **Background**: `#0a0a0a` (Deep Black)
- **Typography**: Bebas Neue (Headlines), Barlow Condensed (Labels), Barlow (Body).
- **Philosophy**: Sharp edges only. No border-radius. High contrast.
